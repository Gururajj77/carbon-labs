Radio buttons are used when a list of two or more options are mutually
exclusive, meaning the user must select only one option.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/radio-button/index.js';
```

<Description markdown={`${cdnJs({ components: ['radio-button'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<c4ai-radio-button-group
  label-position="right"
  orientation="horizontal"
  name="radio-group">
  <c4ai-radio-button label-text="Radio button" value="all"></c4ai-radio-button>
  <c4ai-radio-button
    label-text="Radio button"
    value="cloudFoundry"></c4ai-radio-button>
  <c4ai-radio-button
    label-text="Radio button"
    value="staging"></c4ai-radio-button>
</c4ai-radio-button-group>
```

## Skeleton

For the skeleton variation, utilize `<c4ai-radio-button-skeleton>`.

```html
<c4ai-radio-button-skeleton></c4ai-radio-button-skeleton>
```

## `<c4ai-radio-button-group>` attributes, properties and events

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<c4ai-radio-button-group disabled>`) and `false` means not setting the attribute
(e.g. `<c4ai-radio-button-group>` without `disabled` attribute).
