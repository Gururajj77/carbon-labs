Text fields enable the user to interact with and input data. A single line field
is used when the input anticipated by the user is a single line of text as
opposed to a paragraph.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/text-input/index.js';
```

<Description markdown={`${cdnJs({ components: ['text-input'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<c4ai-text-input
  label="Text input label"
  helper-text="Optional help text"
  invalid-text="Error message">
</c4ai-text-input>
```

## Form item

If the input is part of a form, make sure to import and wrap the input with
`<c4ai-form-item>`

```javascript
import '@carbon/web-components/es/components/form/index.js';
```

<Description markdown={`${cdnJs({ components: ['form'] })}`} />

```html
<c4ai-form-item>
  <c4ai-text-input
    label="Text input label"
    helper-text="Optional help text"
    invalid-text="Error message">
  </c4ai-text-input>
</c4ai-form-item>
```

## `<c4ai-text-input>` attributes and properties

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<c4ai-text-input required>`) and `false` means not setting the attribute (e.g.
`<c4ai-text-input>` without `required` attribute).
